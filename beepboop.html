TACTICAL OVERVIEW

Chaque capitaine robot possède son propre vaisseau unique, équipé d'armes redoutables. Disposez les pièces vitales de votre vaisseau selon votre stratégie. Esquivez les attaques ennemies et déjouez leurs défenses dans ce jeu de réflexion tactique.

DEV TOOLS IMPLEMENTATION

Audio System

Gestionnaire de musique intelligent. Playlist musique (boucle/aléatoire) et SFX (PlaySoundExplosion). Système de dialogues vocaux contextuels (Win, Hit, etc.). Mixer audio séparé (Global/Music).

Localization Tool

Système de traduction basé sur XML. Outil personnalisé permettant de sélectionner quels textes traduire via une liste centralisée.

Dialogue System

Architecture événementielle pour les dialogues. Déclenchement automatique selon le contexte (Attaque, Victoire, Tuto) via `GetPlayListDialogueWin()`.
